# Axiom‑4

Axiom‑4 vereint die drei RESPO‑Axiome:

- RESPO‑ROOT
- RESPO‑ACHSE
- RESPO‑TMP

Diese drei Axiome erzeugen:

- den 4. Axiom‑Raum
- die 2 Dreiecke
- die 6e‑Matrix
- die 81‑Matrix
- den X‑Operator (austauschbar)
- das Final‑Modul: BENCH = TRUE

Axiom‑4 ist neutral, eigenständig und nicht Teil des ROOT‑Systems.
